# Set 和 Map

# Set

Set：ES6 新提供的一种数据结构，类似于数组，但是成员的值唯一、不重复

# Set 基本用法

  1. 构造函数，用来生成Set数据结构
  2. 接收参数：数组或具有iterable接口的其他数据结构
  3. 向Set中添加的值不会发生类型转换，如："5" 和 5 是两个不同的值，注意：Set中NaN === NaN，{} != {}

例子：

let s1 = new Set([1, 2, 2, 2, 3, '3', {}, {}, NaN, NaN])
console.log(s1)  // { 1, 2, 3, '3', {}, {}, NaN }

let s2 = new Set('abcdefgggggfff')
console.log(s2) // { 'a', 'b', 'c', 'd', 'e', 'f', 'g' }

# Set 实例的属性和方法

# 属性

  • size：返回Set实例的成员总数

# 方法

# 操作方法

  • add(value)：添加值，返回Set结构本身，可以连续使用
  • delete(value)：删除值，返回布尔值表示成功与否
  • has(value)：判Set成员是否包含某个值，返回布尔值
  • clear()：清除所有成员，无返回值

例子：

let s = new Set([1, 1, 1, 2, 2, 3])

s.add(4).add(5).add(6)
console.log(s) //  { 1, 2, 3, 4, 5, 6 }

s.delete(2)
console.log(s) //  { 1, 3, 4, 5, 6}

console.log(s.has(2))   // false
console.log(s.size)   // 5

s.clear()
console.log(s)  // {}

# 遍历方法

  • keys()：返回 键名 的遍历器
  • values()：返回 键值 的遍历器
  • entries()：返回 键值对 的遍历器（Set 中键名等于键值）
  • forEach(）：使用回调函数遍历每个成员

注意：Set的遍历顺序就是插入顺序。比如使用Set保存一个回调函数列表，调用时就能保证按照添加顺序调用

例子：

let s = new Set([1, 1, 1, 2, 2, 3])

for (const v of s.keys()) {
  // 依次输出：1，2，3
  console.log(v)
}

for (const v of s.values()) {
  // 依次输出：1，2，3
  console.log(v)
}

for (const v of s.entries()) {
  // 依次输出：[1, 2], [2, 2], [3, 3]
  console.log(v)
}
