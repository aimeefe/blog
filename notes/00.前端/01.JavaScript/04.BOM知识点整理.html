# BOM 浏览器对象模型知识点整理

BOM：浏览器对象模型，它提供了很多对象用于访问浏览器的功能，这些功能与网页内容无关

# window 对象

window：浏览器对象模型（BOM）的核心对象，它表示浏览器的一个实例，具有双重角色：

  • 通过 JavaScript 访问浏览器窗口的接口
  • ECMAScript 的Global对象，在网页中定义的任何对象、变量、函数都以window作为其全局对象

# 使用

# 1. 全局作用域

所有在全局作用域中声明的变量、函数都会变成window对象的属性的方法

var age = 18

function sayAge() {
  // 由于 sayAge() 存在于全局作用域中，因此 this.age 被映射到了 window.age
  console.log(this.age)
}

console.log(window.age === age)  // true
console.log(window.sayAge === sayAge)  // true
sayAge()  // 18

注意：全局变量不能通过 delete 操作符删除，而直接在 window 对象上的定义的属性可以

var age = 18
delete age   // false
console.log(age)  // 18

window.name = 'lily'
delete window.name  // true
console.log(window.name)  // undefined

# window.location

window.location 提供了当前窗口中加载的文档有关的信息，还提供一些导航功能，它既是 window 对象的属性又是 document 对象的属性

document.location === window.location
