# CSS 基础知识汇总

# 00. CSS 盒模型

CSS 盒模型本质上是一个盒子，封装周围的 HTML 元素，它包括：边距（margin），边框（border），填充（padding），和实际内容（content）

盒模型包含标准盒模型、IE 盒模型这两种，它们的区别在于两者 宽高 的计算方式不同：

  1. 标准盒模型的宽高 = content 的宽高
  1. IE 盒模型的宽高 = content + padding + border 的总宽高

# 如何设置这两种盒模型 ？

/* 标准模型 */
box-sizing: content-box;

/* IE 模型 */
box-sizing: border-box;
